중구에서의 범죄를 예측하고 효율적인 순찰 경로를 찾아보세요!
해당 프로그램은 중구의 날씨, 유동인구, 경찰서 위치, CCTV 개수 등
다양한 데이터를 활용하여 각 지점별 범죄를 예측합니다.

범죄의 예측 결과를 바탕으로 순찰 경로를 구성하고,
실시간 교통정보를 반영하여 최적의 경로를 화면에 출력합니다.
예상 초동조치 시간, 경로 순회시간를 확인할 수도 있습니다.

먼저 범죄예측 모델의 학습을 위한 데이터를 DB에 저장했습니다.
저희가 사용한 데이터는 다음과 같습니다.

  • 2014~2019년  기상청 날씨정보
  • 2014~2019년  경찰청 5대 범죄 발생횟수 통계
  • 2014~2019년  경찰청 시간대별 범죄 통계
  • 2021년   행정안전부 CCTV 위치 현황
  • 2021년   경찰청 경찰관서 위치 현황
  • 2021년   SK빅데이터허브 유동인구
  • 2021년   서울특별시 거주인구 통계

우선 현재의 계절, 시간, 날씨 별로 시나리오를 나눠 총 48개의 상황을 가정했습니다.
그 다음, DB의 데이터셋을 이용해 머신러닝ㅡ의사결정나무 모델을 학습시켰습니다.
모델은 각 시나리오에 맞게, 한 변이 100m인 격자로 구분된 지도에서,
해당 범위의 범죄 발생 횟수를 예측했고, 이를 바탕으로 범죄 예측 지도를 완성했습니다.

혼동 행렬(Confusion Matrix)를 활용해 예측 결과를 평가했으며,
준수한 성능을 냄을 확인했습니다.

범죄를 예측한 결과를 이용해 순찰 경로를 구성했습니다.
네이버 지도 API를 이용해 실시간 교통정보에 맞춰 경로를 최적화했습니다.
기존 범죄 데이터를 이용해, 예상되는 범죄 발생 초동조치 시간을 계산했고,
이에 대한 분석을 그래프로 표현했습니다.

이제, 시나리오를 설정하고 현재 가장 좋은 순찰 경로를 찾아보세요!
